# Spam SMS Detection

## Overview
An AI-based Spam SMS Detection system that classifies SMS messages as Spam or Legitimate using Natural Language Processing.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP
- TF-IDF Vectorization
- Naive Bayes Algorithm

## Workflow
1. Data Cleaning
2. Text Preprocessing
3. TF-IDF Feature Extraction
4. Model Training
5. Spam Classification

## Model
Multinomial Naive Bayes

## Result
The model successfully detects spam messages with high accuracy.
